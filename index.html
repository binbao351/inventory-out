Cấu Hình Google Sheet & Apps Script

Hướng dẫn cài đặt Google Apps Script:

  1. Mở file Google Sheet của bạn → Chọn Tiện ích mở rộng (Extensions) → Apps Script.
  2. Dán toàn bộ mã nguồn file Code.gs (đã được tạo sẵn trong thư mục ứng dụng này).
  3. Nhấn Triển khai (Deploy) → Triển khai dưới dạng ứng dụng web (New Deployment → Web App).
  4. Tại mục "Ai có quyền truy cập" (Who has access): Chọn Bất kỳ ai (Anyone).
  5. Coppy URL Web App nhận được và dán vào ô bên trên.
